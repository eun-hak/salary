# 2026년 연봉 실수령액 계산기

2026년 최신 4대보험 요율(국민연금 4.75%, 건강보험 3.595%)과 국세청 간이세액표를 반영한 연봉 실수령액 계산기입니다.

## 기술 스택

- **Next.js 16** (App Router, SSG)
- **TypeScript**
- **Tailwind CSS v4**

## 주요 기능

- 연봉별 실수령액 계산기 (부양가족, 비과세 식대 옵션)
- 연봉 구간별 실수령액 요약 표
- 141개 개별 연봉 상세 페이지 (SSG)
- SEO 최적화 (메타데이터, JSON-LD, sitemap, robots.txt)
- FAQ (FAQPage 구조화 데이터)

## 실행 방법

```bash
npm install
npm run dev    # 개발 서버
npm run build  # 프로덕션 빌드
npm start      # 프로덕션 서버
```
